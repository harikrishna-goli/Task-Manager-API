The starting point is **scaffolding the project structure and environment setup**. Before writing endpoints, you need a clean foundation:

1. **Initialize Git Repository**
   - Create a new repo (`task-manager-api`).
   - Add `.gitignore` for Python, Docker, and environment files.

2. **Environment Setup**
   - Create `.env` and `.env.test` files with DB connection strings and JWT secrets.
   - Define a `config.py` to load settings via Pydantic.

3. **Project Structure**
   - Set up directories: `app/` (API, models, schemas), `tests/`, `alembic/`.
   - Add `docker-compose.yml` with API + DB services.

4. **Dependencies**
   - Install FastAPI, SQLAlchemy, Alembic, psycopg2, python-jose, passlib, pytest.
   - Add a `requirements.txt` or `pyproject.toml`.

5. **Database Base**
   - Define `Base` (SQLAlchemy declarative base).
   - Configure `SessionLocal` for DB connections.

6. **Hello World API**
   - Create `main.py` with a simple FastAPI app.
   - Run it inside Docker to confirm setup works.
